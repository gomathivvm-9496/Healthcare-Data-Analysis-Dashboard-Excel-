# 🏥 Healthcare Data Analysis Dashboard (Excel)

## 📌 Overview
This project analyzes healthcare data using Excel to identify patterns in smoking, weight, diabetes, and medical costs.



## 🎯 Objective
To build an interactive dashboard and extract meaningful healthcare insights.



## 📊 Dataset
Includes:
- Smoker status
- Diabetes status
- Weight category
- BMI
- Charges
- Hospital tier



## ⚙️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers



## 📸 Dashboard Preview
![Dashboard](image/Dashboard.png)



## 🔍 Key Insights
- Smokers show higher health risk patterns
- Medical charges increase with obesity
- Prediabetes group indicates future risk
- BMI varies across hospital tiers



## 💡 Recommendations
- Focus on preventive healthcare
- Monitor high-risk groups
- Improve healthcare access
