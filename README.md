# Healthcare Data Analysis Project
## Overview

This project focuses on analyzing a healthcare dataset to extract meaningful insights about customers, medical charges, and demographic factors. The goal is to perform data cleaning, identify missing values, and understand patterns that impact healthcare costs.
## Objectives 
To analyze healthcare data and identify patterns that can help in predicting outcomes and improving decision-making.
- Clean and preprocess the dataset
- Handle missing values
- Convert data types (especially date columns)
- Calculate customer age
- Analyze healthcare charges based on:
    - Age
    - City tier
    - Hospital tier
- Identify patterns and trends in the dataset



## Dataset Description

The dataset contains the following columns:

- Customer ID - Unique identifier for each customer
- Date of Birth – Used to calculate age
- Children – Number of dependents
- Charges – Medical expenses billed to the customer
- Hospital Tier – Category of hospital (Tier 1, Tier 2, Tier 3)
- City Tier – City classification
- State ID – State identifier
- Age – Calculated using Date of Birth
## Tools Used
- Microsoft Excel
- Data Cleaning Techniques
- Basic Data Analysis Functions
## Data Cleaning Steps
- Removed or handled missing values
- Converted Date of Birth into proper date format
- Fixed errors such as:
    - #VALUE!
    - #REF!
- Standardized column formats
## Key Insights
- Healthcare charges tend to increase with age
- City tier and hospital tier influence medical expenses
- Most customers fall under specific tier categories
- Data inconsistencies can significantly affect analysis
